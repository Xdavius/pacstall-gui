# experimental pacstall-gui
A gui for [Pacstall](https://pacstall.dev) package manager.

## Installation
First you've got to have Pacstall installed, then
`pacstall -I pacstall-gui-git` to install.

## Features
- [x] Install packages
- [x] Themes (light/dark)
- [x] View installation
- [x] gtk3
- [] pacscript details
- [] remove/list pacscripts

A GUI for [Pacstall](https://pacstall.dev) package manager.

## Installation
First you've got to have Pacstall installed, then
`pacstall -I pacstall-gui-git` to install.

## Features
- [x] Install packages
- [x] Themes (light/dark)
- [x] View installation
- [x] gtk3
- [x] remove/list pacscripts
### what needs to be done
- pacscript details
- custom prompts
